It's Saturday and you're posting a pic from a cool new park. The app pops up: “Add location? — Maple Ridge Skate Park.”

How do you share?

  1. Add the exact location so friends can find the spot.
  2. Add only the city and set to Close Friends so it's “kinda” private.
  3. Share with no location (or add it later offline if you really want it).
